# SOP-NC-02 — NACSA Examination Response SOP

| | |
|---|---|
| **Document ID** | SOP-NC-02 | **Version** | 1.0 |
| **Owner / Approver** | CCO + CISO |
| **Parent standard** | [STD-NC-02](../03-standards/STD-NC-02-ncii-audit-standard.md) | **Parent policy** | [POL-23](../02-policies/POL-23-ncii-operational-policy.md) |

## 1. Purpose
Operational response when NACSA initiates an examination of GIBB's NCII compliance.

## 2. Trigger
NACSA notification of upcoming examination — scheduled or ad-hoc.

## 3. Procedure

### Phase 1 — Receipt and stand-up
| # | Actor | Action |
|---|---|---|
| 1 | CCO | Receive NACSA notification; acknowledge per NACSA timeline |
| 2 | CCO | Engage CISO; notify CEO + CRO + General Counsel |
| 3 | CCO | Confirm examination scope, dates, NACSA examiners, documents requested |
| 4 | CISO | Assemble examination response team (technical SMEs) |
| 5 | CCO + CISO | Brief executive team |

### Phase 2 — Preparation
| # | Actor | Action |
|---|---|---|
| 6 | CISO + CCO | Pre-examination document assembly per NACSA list |
| 7 | Internal Audit | Provide relevant NCII audit findings + corrective action status from [REG-NCA](../06-registers/REG-NCA-ncii-audit-findings-register.md) |
| 8 | CISO | Confirm NCII operational documents are current (CRMF, NCIIF, POL-23, STD-NC-01, STD-NC-02, [REG-NCN](../06-registers/REG-NCN-nacsa-notification-register.md)) |
| 9 | CCO | Coordinate logistics (rooms, system access for examiners, evidence demonstration capability) |

### Phase 3 — Examination conduct
| # | Actor | Action |
|---|---|---|
| 10 | CISO + technical SMEs | Walk examiners through requested areas; demonstrate controls |
| 11 | CCO | Single-point-of-contact for examiner; daily check-ins |
| 12 | General Counsel | Available for sensitive questions |
| 13 | Examination response team | Document all examiner requests, responses provided, follow-up commitments |

### Phase 4 — Closure and follow-up
| # | Actor | Action |
|---|---|---|
| 14 | CCO + CISO | Exit briefing with NACSA; understand preliminary findings |
| 15 | CCO | Receive formal NACSA findings report |
| 16 | CISO + CCO | Develop remediation plan for findings within NACSA-specified timeline |
| 17 | CCO | Record findings in [REG-NCA](../06-registers/REG-NCA-ncii-audit-findings-register.md); corrective actions in [REG-CAP](../06-registers/REG-CAP-corrective-action-register.md) |
| 18 | CCO | Communicate findings outcome to RMC + Board |
| 19 | Action owners | Track remediation to closure; report to NACSA per timeline |
| 20 | CCO | Final confirmation to NACSA on remediation completion |

## 4. Confidentiality
Examination scope, findings, evidence handled as Internal — restricted. Public disclosure only with NACSA + GIBB CEO approval.

## 5. Evidence
NACSA notification; document assembly inventory; examination meeting notes; findings report; remediation plan; closure confirmation. Permanent retention.
